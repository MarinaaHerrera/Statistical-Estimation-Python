# Statistical Estimation & Predictive Analytics

**Business Objective:**
To apply statistical estimation techniques to demographic data, deriving confidence intervals and hypothesis testing to inform data-driven decision making. 

**Tech Stack:** Python (SciPy, Pandas, NumPy)

**Core Competencies Demonstrated:**
* Point and Interval Estimation
* Hypothesis Testing 
* Statistical Data Transformation

**Repository Structure:**
* `statistical_estimation_analysis.ipynb`: The core Python notebook detailing the statistical mathematical logic and data visualization.
* `child_outlook_dataset.csv`: The demographic dataset used for the analysis.
